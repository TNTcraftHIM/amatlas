Amatlas UI Skin Gallery

这是普通文字冒险/跑团页面 chrome 的换皮选材页:同一套 `#mapname/#place/#scene/#look/#choices/#status` 和插件工具栏,只通过 HTML/CSS 模板切换外观。它不是 demo,不是新模块,也不新增 world/View 字段。八套 skin 覆盖干净基线、纸页、终端、档案、脏 zine、神秘学边注、霓虹夜城和野外笔记。

边界

Gallery 用真实 Amatlas.boot 跑一个小 world,只是为了让按钮、锁定提示、状态栏、小地图、存档和成就都出现在同一页里。

切 skin 只改 document.documentElement.dataset.ui;运行时、模块、核心契约都不参与。

正式游戏可链接 engine/ui/amatlas-skins.css 或让 build 内联这份共享 CSS;不要给 world 增加 theme/skin 字段。
